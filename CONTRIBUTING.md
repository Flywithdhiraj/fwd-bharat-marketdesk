# Contributing

Thanks for your interest in FWD Bharat MarketDesk.

## Useful areas

- Dhan market-data reliability and rate-limit handling
- NSE/BSE scanner quality
- Options analytics and chart-review workflows
- Accessibility and Windows desktop packaging
- Test coverage for read-only safety guarantees

## Local checks

Run these before proposing changes:

```powershell
npm ci
npm run check
```

Do not add generated build output, market-data caches, credentials, screenshots, or `node_modules` to git.
