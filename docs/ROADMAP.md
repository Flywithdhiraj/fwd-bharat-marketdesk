# Roadmap

## Near term

- Add screenshots and short demo clips for the main scanner, Options Hub, and chart review workflows.
- Add GitHub Actions status checks to every pull request.
- Improve onboarding docs for DhanHQ credential setup without exposing secrets.
- Add fundamentals import/provider support for ROCE, ROE, debt/equity, PE, EPS growth, and sector metadata.

## Safety and security

- Keep all broker execution manual-only.
- Expand regression tests around IPC boundaries, credential storage, and disabled order actions.
- Review dependency and packaging risk before every tagged release.

## Community

- Label beginner-friendly documentation and test issues.
- Keep feature requests tied to read-only data, analysis, accessibility, or documentation improvements.
- Document architecture decisions that help contributors work safely.
